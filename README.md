# calendar 📆

Personal project for a replacement for Google Calendar, so similar in style and with the features I use.

## Planned features

- calendar options
  - Share
    - Share read only link
    - Share full edit link
- able to delete one particular recurrence, or from one day forward
- able to drag and move an event forward/back a few days
- ical import/export
- scrolling left and right should have same effect as scrolling up and down (changing the current month)
- on mobile, when clicking a day with already some events, open a daily view; a further click then opens the new event modal
- deployment to Oracle Cloud (Always Free VPS), Mongo Atlas free tier
